Clone of mobile game Build Stack

Using Unity 3D

Build stack is game where you build a stack, thats it, see how high you can go. We start of with two spawners on the left and right sides of the screen which alternate spawning slabs. These slabs move towards the center stack and the player's job is to line them up with the stack as best they can in order to create the tallest stack possible. To start right click or press spacebar and then again to stop and place the current slab. Any over hang gets cut off and all future slabs are resized to the size of the remaining peice. The game ends when any slab is stopped and misses the main stack.

This is a simpler project that mostly just solidified my confidence in using the Unity editor and having my perferred workflow. The most challenging part was the math for spawning both a new remaining and overhang piece after cutting the slab. I did learn about gizmos and TextMeshPro but mostly this was a programming side heavy project.

At the start a cube/slab prefab with if own moving script is instantiate. Every slab knows which direction it is moving by knowing from which of the two spawner it came. The current moving slab also handle the player's input and handles the logic of slicing the slab into a remaining and hangover peices. The game diffculty increases by faster moving slabs every new slab up until slab forty. Going past of stopping before the stack ends the game and trigger a restart option.

Other little quality of live things include adjusting the camera height so that the stack is always in center view once it starts getting too tall, UI using TextMeshPro is used for current and highscore much like in previous Flappy Bird project, and gizmos highlight the spawan points.
