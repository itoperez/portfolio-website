Projects in AR Bundle

Using Unity AR

This is a collection of projects in Unity AR. The first project uses Image Tracking to find the N64 cartridge and projects Link and Zelda onto it with a little bit of shader fun and a transition script. In the next video I used Surface Tracking to place a music box that plays Edelweiss when placed on a flat surface (it will face you when placed and has 3D audio). The last project in the third video is Unity's own tutorial on Face Tracking. I mostly stuck to their project and learned how to create a face tracking mask app on Android.

The first Link and Zelda was kept on PC and used a webcam to find the image. The image tracking was done with Vuforia which is now a bit outdated but I learn a lot regardless. The next two projects used AR Core and were built on Android. Below will have three links to the three project repositories.
