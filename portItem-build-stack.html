Clone of mobile game Build Stack

A hyper-casual game

Two spawners alternate spawning slabs towards the center and the player attempt to build the highest stack possible. To start press Fire1 or spacebar and do the same to place stop slabs and place them. Any slab over hang gets cut off and any future slabs are resized to the size of the remaining peice. The game ends when any slab is stopped and misses the main stack.

The core script, MovingCube, attached to every instantiate cube. This script is responsible for managing movement and player input. It handles the logic of slicing the slab into a remaining and hangover peice. Increases game diffculty and triggers OnDied event used for Game Over/restarting.

Other supporting scripts include a GameManager to keep track of if player isAlive and where the slabs are spawning from. CubeSpawner to spawn the slabs and MainCamera to adjust camera height to keep the game in view. A ScoreText script in designed to keep track of current score and all time high score.
