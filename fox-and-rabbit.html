Mini World Fox and Rabbit

An Interactive 3D World

Fox and Rabbit started out as a simple assignment to create a 3D interactable world with at least three different interactions. Besides the core assignment requirements I really wanted to test myself here and see how much I could get done in just one day so this project became a bit of a mini game jam. All assest were free or from previous projects, all of the code if from scratch (except the fireflies flight movement, from a previous project), and everything was completeted in 21 hours.

Although this was a short project I wanted to make the game feel as complete as possible so I spent most of my time in level design. I imported the terrain from an old assignment and use the trees terrain tool to paint trees in. The terrain was baked and I added a Nav Mesh for the rabbits I would release which have there Nav Mesh Agent and script to wander around the playable area. The plants and rocks were individually placed and the skybox is custom made. Ambient sounds were added and fireflies placed to make the world fee alive (another assignment requirement). The fox and rabbit were imported separately and after setting up the models with their textures I made sure their animations played when they were supposed to and added a character controller for the fox.

Now that the main world is set, I had to rush a bit with the interaction. I first added a chest that opens when you approach, then flowers that needed to be smelled, rabbits to be freed from crates, and mushrooms that you had to jump on like a trampoline. All interactable objects have sounds to let the player know that they have interacted with them correctly. To keep track of the interactions need to unlock the ultimate goal behind the electric fence (lighting from an inported package) I made a simple UI accessible by pressing the ESC key. Overall this project was really fun and solidified my confidence in being able to create a completed something from start to finish on my own.
